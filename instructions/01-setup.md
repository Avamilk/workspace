# FPL App Setup Instructions

Initialize Next.js 14 project:
- npx create-next-app@latest fpl-app --typescript --tailwind --app
- cd fpl-app
- npm install @fontsource/vt323 lucide-react framer-motion

Create folders:
- mkdir -p app/components/ui app/components/pitch app/components/ai
- mkdir -p lib/ai lib/fpl

Test: npm run dev

Report back when localhost:3000 loads.
